getavgbsqprice

NAME
----
getavgbsqprice - get average BSQ price in btc and usd

SYNOPSIS
--------
getavgbsqprice
    --days=<30|90>

DESCRIPTION
-----------
Returns the volume weighted average BSQ trade price over n days,
in BTC and USD.

EXAMPLES
--------
$ ./bisq-cli --password=xyz --port=9998 getavgbsqprice --days=30
